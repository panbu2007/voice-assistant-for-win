# voice-assistant-for-win

## 简介
一款基于科大讯飞语音技术的Windows语音助手

## 功能
界面：
- [x] 悬浮窗
- [x] 弹幕显示
- [x] 最小化至系统托盘

系统：
- [ ] 语音唤醒（待上传）
- [x] 语音应答
- [x] 打开应用
- [x] 大语言模型对话
- [x] 音量调节
- [x] 亮度调节

## 安装与使用
建议python>=3.8
```
pip install -r requirements.txt
```
注意：由于使用科大讯飞语音技术，需要在`config.py`中输入对应的接口认证信息
